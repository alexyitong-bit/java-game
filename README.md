# JavaFX Battle Game 🎮

一款用JavaFX开发的回合制RPG战斗游戏，适合学习JavaFX游戏开发。

## 功能特性

✨ **核心系统**
- 完整的角色系统（玩家、敌人、属性）
- 回合制战斗系统
- 技能和魔法系统
- 伤害计算和状态效果
- 动画特效
- 战斗日志

🎯 **游戏流程**
1. 主菜单
2. 选择角色（战士、法师、弓箭手）
3. 选择敌人（哥布林、兽人、龙、黑骑士）
4. 开始战斗
5. 轮流操作（使用技能、防守、道具）
6. 战斗结束和胜负判定

## 项目结构

```
src/main/java/com/javagame/
├── Main.java                    # 程序入口
├── model/                       # 数据模型
│   ├── Character.java          # 角色基类
│   ├── Player.java             # 玩家类
│   ├── Enemy.java              # 敌人类
│   └── Skill.java              # 技能类
├── game/                        # 游戏逻辑
│   ├── BattleSystem.java       # 战斗系统
│   └── GameData.java           # 游戏数据
└── ui/                          # 用户界面
    ├── MainMenuScene.java      # 主菜单
    ├── CharacterSelectScene.java # 角色选择
    ├── EnemySelectScene.java   # 敌人选择
    ├── BattleScene.java        # 战斗界面
    └── ResultScene.java        # 结果界面
```

## 运行方法

### 前置要求
- Java 17+
- Maven 3.6+

### 编译和运行

```bash
# 编译项目
mvn clean compile

# 运行游戏
mvn javafx:run

# 或打包成JAR
mvn clean package
java -jar target/javafx-battle-game-1.0-SNAPSHOT.jar
```

## 游戏说明

### 角色选择
- **战士**：HP 150，防御 25，物攻 18 - 高HP和防御，低速度
- **法师**：HP 100，魔攻 25，魔防 12 - 高魔法，低防御
- **弓箭手**：HP 120，物攻 20，速度 16 - 高速度和物攻，中等防御

### 敌人类型
- **哥布林**：简单敌人，HP 40，金币奖励 50
- **兽人**：中等敌人，HP 80，金币奖励 100
- **龙**：困难敌人，HP 150，金币奖励 300
- **黑骑士**：困难敌人，HP 120，金币奖励 250

### 战斗操作
- **攻击**：普通物理攻击
- **技能**：消耗魔力的特殊攻击（不同角色有不同技能）
- **防守**：减少下一轮伤害50%
- **道具**（后续版本）：使用恢复道具

## 角色技能

### 战士技能
1. 斩击 - 物理攻击，伤害25，消耗魔力10
2. 盾挡 - 强化防御，消耗魔力15
3. 战吼 - 魔法攻击，伤害15，消耗魔力20
4. 必杀斩 - 终极技能，伤害50，消耗魔力30，冷却3回合

### 法师技能
1. 火球术 - 魔法攻击，伤害30，消耗魔力15
2. 冰冻术 - 魔法攻击，伤害20，消耗魔力20，冷却1回合
3. 治疗术 - 恢复HP，消耗魔力25
4. 魔法风暴 - 终极技能，伤害60，消耗魔力40，冷却3回合

### 弓箭手技能
1. 箭射 - 物理攻击，伤害20，消耗魔力8
2. 连续射击 - 物理攻击，伤害35，消耗魔力15，冷却1回合
3. 闪避 - 躲避下一次攻击，消耗魔力10
4. 致命一箭 - 终极技能，伤害55，消耗魔力30，冷却2回合
